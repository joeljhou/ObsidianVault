主要官方来源：[Maven官网](https://maven.apache.org) ｜ [Github](https://github.com/apache/maven)

> Maven 是一个项目管理和构建工具，主要用于 Java 项目的依赖管理、编译、测试、打包和部署。它采用 **POM（Project Object Model）** 作为项目配置文件（`pom.xml`），并提供了一套标准化的项目构建流程。
